# Decision PRO — Documentation

## Structure

| Path | Purpose |
|------|---------|
| **docs/guides/** | Living guides: testing, auth, UI, quick start, etc. Use these for day-to-day work. |
| **docs/archive/** | Historical status reports, fix summaries, and completion notes. Kept for reference. |
